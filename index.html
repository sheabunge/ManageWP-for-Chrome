ManageWP for Chrome

Note: ManageWP is now an official app in the Chrome Web Store! All future updates will be posted there.

Provides a link to your ManageWP Dashboard from Chrome's new tab page.

More information at the project's website. Download, fork, or watch the project on GitHub.

Updates will be posted on my blog.

Installation

  1. Download a zip of the master branch to your local machine
  2. Unzip the file
  3. Launch Chrome
  4. Click the wrench icon on the browser toolbar.
  5. Select Tools > Extensions.
  6. Locate the managewp-for-chrome.crx file on your computer and drag the file onto the Extensions page.
  7. Confirm the installation

If you're having trouble, please create an issue.
